## Сущность пользователя

### Public api
    
Types:
  - `User` – тип описывающий пользователя

Selectors:
  - `getAuthData` – селектор для получения авторизированного пользователя
  - `getUserInited` – селектор для получения информации об авторизации пользователя
  - `getUserRoles` – селектор для получения ролей пользователя
  - `isUserAdmin` – селектор для проверки роли Admin у пользователя
  - `isUserManager` – селектор для проверки роли Manager у пользователя
